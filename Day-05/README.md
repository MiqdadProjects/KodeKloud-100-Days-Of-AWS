# Day 05 - Create GP3 Volume

**YouTube Video Link:**  
[Watch the solution here](https://www.youtube.com/watch?v=YhCbpobVnfA)

**Task Summary:**  
In this task, you learn how to create a GP3 (General Purpose SSD v3) EBS volume in AWS, which is the latest and most cost-effective general-purpose block storage option for EC2 instances. The video covers the steps to provision a new GP3 volume, configure its size and IOPS, and understand when to choose GP3 over other EBS volume types. EBS volumes are the primary block storage solution for AWS EC2 instances.

**What you will learn:**  
- What an AWS EBS (Elastic Block Store) volume is and the difference between GP2 and GP3 types.  
- How to create a GP3 volume using the AWS Management Console.  
- How to configure volume size (GiB), IOPS (up to 16,000), and throughput (up to 1,000 MiB/s) independently.  
- Best practice: GP3 is more cost-efficient than GP2 — prefer GP3 for new workloads as it provides baseline 3,000 IOPS at no extra cost.

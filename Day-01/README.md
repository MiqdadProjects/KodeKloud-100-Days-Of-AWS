# Day 01 - Create Key Pair

**YouTube Video Link:**  
[Watch the solution here](https://www.youtube.com/watch?v=v8p7aZbGQ-4)

**Task Summary:**  
In this task, you learn how to create an EC2 Key Pair in AWS, which is essential for securely connecting to EC2 instances via SSH. The video walks through the process of generating a key pair using the AWS Management Console and understanding the different key formats available. This is one of the foundational steps before launching any EC2 instance on AWS.

**What you will learn:**  
- What an AWS EC2 Key Pair is and why it is needed for SSH access.  
- How to create a Key Pair using the AWS Management Console.  
- The difference between `.pem` (used with OpenSSH) and `.ppk` (used with PuTTY) formats.  
- Best practice: Download and securely store your private key immediately — AWS does not store it after creation.

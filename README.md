Welcome to my **100 Days of AWS Cloud Challenge** repository! 🚀

In this repository, I am documenting my daily journey of solving **KodeKloud Engineer AWS tasks**. Cloud networking and infrastructure can seem overwhelmingly cryptic at first, but tackling one hands-on practical implementation each day transforms the complex into the achievable.

Each day features a detailed **README** breaking down the "Why" and "How" of the task, along with a linked **YouTube video explanation** to help visualize the steps!

## 📌 What's Inside?

- 🏗️ **KodeKloud Engineer Solutions:** From basic EC2 provisioning to complex VPC Peering and serverless architectures.
- 📖 **Step-by-Step Breakdowns:** I document the exact CLI commands, GUI clicks, and scripts needed to succeed.
- 💡 **Real-World Contexts:** "Why do we use an ALB instead of an NLB here?" - you'll find the practical answers inside.
- 🎓 **Beginner to Advanced:** The tasks scale up in difficulty organically as the days progress!

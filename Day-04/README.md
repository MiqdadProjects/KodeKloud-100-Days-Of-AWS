# Day 04 - Enable Versioning for S3 Bucket

**YouTube Video Link:**  
[Watch the solution here](https://www.youtube.com/watch?v=bK-EOPipqMs)

**Task Summary:**  
In this task, you learn how to enable versioning on an AWS S3 bucket, a feature that keeps multiple versions of an object so you can recover from accidental deletions or overwrites. The video walks through navigating to an existing S3 bucket's settings and enabling the versioning option. S3 versioning is a key data protection strategy for any production environment.

**What you will learn:**  
- What S3 versioning is and why it is important for data recovery.  
- How to enable versioning on an existing S3 bucket via the AWS Console.  
- How versioned objects behave — each upload creates a new version with a unique Version ID.  
- Best practice: Enable versioning in combination with S3 Lifecycle policies to manage storage costs for older versions.

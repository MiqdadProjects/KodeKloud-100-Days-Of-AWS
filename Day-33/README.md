## 🧠 Task Overview

Learn the basics of serverless computing by creating a simple AWS Lambda function from the AWS Management Console.

---

## 🎯 Key Takeaways & Best Practices

- ⚡ **Serverless Efficiency:** Focus completely on your code. AWS manages all server provisioning, scaling, and patching.
- 💰 **Pay-Per-Use:** You only pay for the exact compute time your code consumes (down to the millisecond).

---

## 💡 Real-World Scenario

> **Simple Cron Jobs:** Easily replace legacy crontab scripts running on dedicated VMs with a lightweight Lambda function paired with EventBridge.

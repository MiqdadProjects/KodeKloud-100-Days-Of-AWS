**Task Summary:**  
In this task, you learn how to create and attach an Elastic Network Interface (ENI) to an EC2 instance in AWS. An ENI is a virtual network card that can carry its own private IP, Elastic IP, MAC address, and security group settings — and can be moved between instances. This is a powerful feature for building highly available network configurations.

**What you will learn:**  
- What an Elastic Network Interface (ENI) is and how it differs from the default primary network interface.  
- How to create a standalone ENI and attach it to a running or stopped EC2 instance via the AWS Console.  
- Use cases for ENIs: high availability failover, dual-homed instances, and management network separation.  
- Best practice: ENIs can be detached and reattached to other instances, making them useful for quickly shifting network identity (IP/MAC) during failover scenarios.

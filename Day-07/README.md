# Day 07 - Change EC2 Instance Type

**YouTube Video Link:**  
[Watch the solution here](https://www.youtube.com/watch?v=On3gvj7CVNo)

**Task Summary:**  
In this task, you learn how to change the instance type of an existing EC2 instance in AWS, which allows you to scale your compute resources up or down based on your workload needs. The video demonstrates how to stop the instance, modify its instance type, and restart it — a key operational skill for managing cost and performance on AWS. This is an important concept for right-sizing your EC2 instances.

**What you will learn:**  
- Why and when you would need to change an EC2 instance type (scaling up/down).  
- How to stop an EC2 instance, change its instance type, and restart it via the AWS Console.  
- Understanding instance type families (e.g., `t3`, `m5`, `c5`) and their use cases.  
- Best practice: Always stop the instance before changing its type; monitor CPU and memory utilization with CloudWatch before making sizing decisions.

# Day 03 - Create Subnet

**YouTube Video Link:**  
[Watch the solution here](https://www.youtube.com/watch?v=UStuUoop4WY&t)

**Task Summary:**  
In this task, you learn how to create a Subnet within an AWS VPC (Virtual Private Cloud), which allows you to segment your network into smaller, isolated ranges. The video covers how to choose an Availability Zone, define a CIDR block, and associate the subnet with a VPC. Subnets are a core building block of AWS networking and are essential for organizing resources securely.

**What you will learn:**  
- What a Subnet is and how it fits within an AWS VPC architecture.  
- How to create a public or private subnet using the AWS Management Console.  
- Understanding CIDR block notation (e.g., `10.0.1.0/24`) for IP range allocation.  
- Best practice: Use separate subnets for public-facing and private resources to improve security.

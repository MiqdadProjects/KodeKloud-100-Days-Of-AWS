# Day 10 - Attach Elastic IP to EC2 Instance

**YouTube Video Link:**  
[Watch the solution here](https://www.youtube.com/watch?v=CKd9LqGeRlY)

**Task Summary:**  
In this task, you learn how to allocate and attach an Elastic IP (EIP) address to an EC2 instance in AWS. An Elastic IP is a static, public IPv4 address that remains associated with your account and can be remapped to any instance at any time. This is essential for hosting applications that require a consistent public endpoint even after instance restarts.

**What you will learn:**  
- What an AWS Elastic IP address is and why it differs from a regular dynamic public IP.  
- How to allocate a new Elastic IP from the AWS console and associate it with a running EC2 instance.  
- How to disassociate and release an Elastic IP to avoid unnecessary charges.  
- Best practice: Always release unused Elastic IPs — AWS charges for EIPs that are allocated but not associated with a running instance.
